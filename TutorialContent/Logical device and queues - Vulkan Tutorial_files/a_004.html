                                      Rafael Beckel 3 years ago edited

                                      For those using MacOS 10.13 (High Sierra), compilation will fail because Apple have not implemented optional::value() yet, so you either have to update to 10.14 (Mojave) or do this workaround:


                                      struct QueueFamilyIndices {
                                          std::optional<uint32_t> graphicsFamily;

                                          bool isComplete() {
                                              return graphicsFamily.has_value();
                                          }

                                          uint32_t value() {
                                              if(!graphicsFamily.has_value())
                                                  throw std::logic_error("bad optional access");
                                              return *graphicsFamily;
                                          }
                                      };

                                      And then call it as indices.value(); instead.
