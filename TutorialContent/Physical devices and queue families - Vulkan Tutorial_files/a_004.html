                                        Daddoggetty Alexander Overvoorde 3 years ago edited

                                        Great Tutorial Alexander,

                                        Unfortunately I am using the SteamOS desktop (debian Jessie) so my g++ is stuck at C++11, but the vulkan drivers are installed and working.
                                        As C++11 doesn't have std::optional function I created a template class "Optional" and placed it in a file "optional.h".

                                        This allows minimal changes to your code and easy update once my g++ gets an update to C++17:

                                        At the top of the code I replaced:
                                        #include <optional> with #include "optional.h"

                                        In the structure QueueFamilyIndices I replaced:
                                        std::optional<uint32_t> graphicsFamily; with
                                        Optional<uint32_t> graphicsFamily;

                                        The "optional.h" file is:

                                        #ifndef _OPTIONAL_H_
                                        #define _OPTIONAL_H_

                                        // TEMPLATE CLASS:
                                        // Class to create an object to 
                                        // store a value of any type.

                                        // Declaration
                                        template <class T> 
                                        class Optional
                                        {
                                            private:  
                                                  T element;
                                                  bool element_exists;
                                            
                                            public:
                                                  Optional();            // constructor
                                                  ~Optional();           // destructor
                                                  void operator = (T );  // assignment
                                                  bool has_value();      
                                                  T value();           
                                        };

                                        // PUBLIC methods

                                        // constructor
                                        template <class T>
                                        Optional<T>::Optional()
                                        {
                                              element_exists=false;   
                                        }

                                        // destructor
                                        template <class T>
                                        Optional<T>::~Optional()
                                        {  
                                        }

                                        // assignment operator overide
                                        template <class T>
                                        void Optional<T>::operator = (T val)
                                        {
                                              element = val;
                                              element_exists = true;
                                              return;
                                        }

                                        // Method to check for a valid value
                                        template <class T>
                                        bool Optional<T>::has_value()
                                        {
                                              if (element_exists) 
                                              {
                                                      return true;
                                              }
                                              return false;    
                                        }


                                        // Method to return the value
                                        template <class T>
                                        T Optional<T>::value()
                                        {
                                              return element;
                                        }

                                        #endif

                                                                              Alain Mosnier a year ago edited

                                                                              This tutorial looks like good help to get started with Vulkan, which is really dense, and I am thankful for that, but I have to react on the advice given in this article to:

                                                                              give each device a score and pick the highest one

                                                                              You do not need that extra "score" concept, which does not exist in reality.

                                                                              What you really are trying to achieve here is to create a "strict order" between devices (and their queue families). While a number allows you to do that (the score), it is much better to implement the strict order directly, exactly matching your required semantics.

                                                                              Basically, you should define the necessary comparison operators (like operator<). For instance, if device1 has a queue family with the graphics bit on, while device2 does not have it, then device2 < device1.

                                                                              In order to implement that logic, you either overload the comparison operators, or you define you own functions (less_than for instance). You can then for instance use std::max to pick the best device. std::max has an optional comp parameter which basically is a less_than function.

                                                                              Example with Vulkan-Hpp types:

                                                                              bool less_than(vk::QueueFlags lhs, vk::QueueFlags rhs)
                                                                              {
                                                                              	/* For now, just care about graphics. */
                                                                              	if (rhs & vk::QueueFlagBits::eGraphics && !(lhs & vk::QueueFlagBits::eGraphics))
                                                                              		return true;
                                                                              	return false;
                                                                              }

                                                                                                                                                    JuanPatatas 3 years ago edited

                                                                                                                                                    On Ubuntu 16.04 it is required to install gcc-7 packages(c++17) just for use <optional> headers.
                                                                                                                                                    Even if you try to fix it with <experimental/optional>, the test ends with "failed to find a suitable GPU!" message, because Vulkan also needs c++17 to work properly.

                                                                                                                                                    jlblancoc shows how to do it (https://gist.github.com/jlblancoc/99521194aba975286c80f93e47966dc5).

                                                                                                                                                    Install the gcc-7 packages:
                                                                                                                                                    sudo apt-get install -y software-properties-common
